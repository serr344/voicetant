# Project Purpose

Voicetant is a fast desktop AI voice assistant that stays at the top of the screen for instant access. The user speaks to the application, the speech is converted to text (STT), processed by an LLM, and then converted back into speech (TTS) as a response. It is designed to provide quick, natural, hands-free interaction for solving problems, asking questions, and getting real-time assistance.
